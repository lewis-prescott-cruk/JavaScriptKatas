## JavaScript Roman Numerals Kata

### Dependencies
- yarn / npm install

### Run tests
- npm test

### The Rules
- Working in pairs
- TDD approach (Code can only be written with a failing test)
- Red, Green, Refactor
- Output via the test command only

### The Task
1. Remove unnecessary tests
2. Refactor code as you wish
3. Add scenarios for:
    - 9
    - 10
    - 11
    - 14
    - 15
    - 19
    - 20

#### Roman Numerals
I = 1  
IV = 4  
V = 5  
V1 = 6  
IX = 9  
X = 10  
XIV = 14  
XV = 15  
XIX = 19  
XX = 20
